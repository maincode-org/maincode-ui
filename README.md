## Install

```bash
npm install --save maincode-ui
```

## Usage

Made to be used with Ionic Framework!

```tsx
import React, { Component } from 'react';

import MyComponent from 'maincode-ui';
import 'maincode-ui/dist/index.css';

class Example extends Component {
  render() {
    return <MyComponent />;
  }
}
```
